# Input

The application does not receive any data or interaction from the users. It is made up of the pre-coded information that includes the students' name, degree program, and school name. This information is coded in Text components, while the style information like the yellow background is contained in StyleSheet. 

# Process

Once the app is run, React Native will render the interface based on the code in the source file along with the applied styles. In this case, one View component will be used as the wrapper for all of the information, while the other three Text components will be responsible for displaying this information. StyleSheet will be used for the consistent formatting of the components, including the yellow background. The app is compiled and executed using Expo.

# Output

The result is a simple interface of the mobile application that shows the name of the student, the degree program, and the school. The text has the yellow background as it is specified in the requirements for the assignment.
